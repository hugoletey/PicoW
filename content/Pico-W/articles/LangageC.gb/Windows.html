CMake

VS Build Tools

Download the Build Tools version of your Visual Studio from: https://visualstudio.microsoft.com/fr/downloads/
You'll find it under "Outils pour Visual Studio."
Ensure that you have all five checkboxes ticked as shown in the image before starting the download.

Python

Download the latest version from: https://www.python.org/downloads/windows/
Make sure to check all options on the "Optional Features" page and select "Add Python to environment variables."

Git

Download the latest version from: https://git-scm.com/downloads
Ensure to select the same options as shown in the images.

Setting Up the Environment

Create a directory named "Pico" at the root of your C drive.

Open your command prompt and navigate to your directory: cd Pico

Download the Pico-sdk file with the command: git clone -b master https://github.com/raspberrypi/pico-sdk.git

Go into the pico-sdk directory: cd pico-sdk and run the command: git submodule update --init

Once the files are downloaded, go back: cd .. and download the code examples with the command: git clone -b master https://github.com/raspberrypi/pico-examples.git

Open your Visual Studio Installer and launch a terminal by clicking the "Launch" button in the Build Tools section. Once on the terminal, type code to open Visual Studio.

Download the CMake Tools extension on Visual Studio, then go to settings, click on extensions, then CMake Tools. Scroll down to CMake: Configure Environment and type: Pico_SDK_PATH in ITEM and ..\..\pico-sdk in value. In CMake: Generator, type NMake Makefiles.

You can now code in C using Visual Studio.
