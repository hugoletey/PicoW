CMake

Téléchargez la dernière version depuis : https://cmake.org/download/

Outils de build de Visual Studio (Build Tools)

Téléchargez la version Build Tools de votre Visual Studio depuis : https://visualstudio.microsoft.com/fr/downloads/
Vous la trouverez sous "Outils pour Visual Studio".
Assurez-vous de cocher les cinq cases comme illustré dans l'image avant de démarrer le téléchargement.

Python

Téléchargez la dernière version depuis : https://www.python.org/downloads/windows/
Assurez-vous de cocher toutes les options sur la page "Fonctionnalités optionnelles" et de sélectionner "Ajouter Python aux variables d'environnement".

Git

Téléchargez la dernière version depuis : https://git-scm.com/downloads
Assurez-vous de sélectionner les mêmes options que celles illustrées dans les images.

Configuration de l'Environnement

Créez un répertoire nommé "Pico" à la racine de votre disque C.

Ouvrez votre invite de commande et naviguez jusqu'à votre répertoire : cd Pico

Téléchargez le fichier Pico-sdk avec la commande : git clone -b master https://github.com/raspberrypi/pico-sdk.git

Accédez au répertoire pico-sdk : cd pico-sdk et exécutez la commande : git submodule update --init

Une fois les fichiers téléchargés, revenez en arrière : cd .. et téléchargez les exemples de code avec la commande : git clone -b master https://github.com/raspberrypi/pico-examples.git

Ouvrez votre Installateur Visual Studio et lancez un terminal en cliquant sur le bouton "Lancer" dans la section Outils de build. Une fois sur le terminal, tapez code pour ouvrir Visual Studio.

Téléchargez l'extension Outils CMake sur Visual Studio, puis allez dans les paramètres, cliquez sur extensions, puis sur Outils CMake. Faites défiler jusqu'à Environnement de configuration CMake et saisissez : Pico_SDK_PATH dans ITEM et ..\..\pico-sdk dans value. Dans Générateur CMake, saisissez NMake Makefiles.

Vous pouvez désormais coder en C à l'aide de Visual Studio.
