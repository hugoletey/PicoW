GPIO Programming

Introduction to GPIO:

  1. GPIO Pins: GPIO pins are the digital input/output pins on a microcontroller that can be independently controlled to either read digital signals (input) or output digital signals (output).
  2. Pin Numbering: GPIO pins are typically numbered for easy reference. On the Raspberry Pi Pico, GPIO pins are numbered from GP0 to GP28.

Controlling Digital Outputs and Reading Digital Inputs:

  1. Digital Outputs:
    • To set a GPIO pin as an output, you use a function such as pin.value(1) or pin.high() to set the pin to HIGH (3.3V) or pin.value(0) or pin.low() to set it to LOW (0V).
    • Example (MicroPython): 
      from machine import Pin
import utime

led_pin = Pin(25, Pin.OUT)

while True:
    led_pin.toggle()
    utime.sleep(1)

      In this example, the LED connected to GPIO pin 25 will toggle on and off every second.

  2. Digital Inputs:
    • To read the state of a GPIO pin configured as an input, you use pin.value() method.
    • Example (MicroPython): 
      from machine import Pin
import utime

button_pin = Pin(12, Pin.IN, Pin.PULL_DOWN)

while True:
    if button_pin.value() == 1:
        print("Button pressed!")
    utime.sleep(0.1)

      In this example, the program checks if the button connected to GPIO pin 12 is pressed. If pressed, it prints a message.

PWM (Pulse Width Modulation) for Controlling Analog Devices:

  1. PWM in MicroPython:
    • MicroPython provides PWM support through the machine.PWM class.
    • Example (MicroPython): 
      from machine import Pin, PWM

pwm_pin = Pin(15)
pwm = PWM(pwm_pin)

pwm.freq(1000)  # Set PWM frequency to 1kHz
pwm.duty_u16(32768)  # Set duty cycle to 50%

      In this example, we create a PWM object on GPIO pin 15, set the frequency to 1kHz, and set the duty cycle to 50%.

By mastering GPIO programming and PWM, you can control a wide range of digital and analog devices with your microcontroller, enabling you to build diverse projects and applications.
