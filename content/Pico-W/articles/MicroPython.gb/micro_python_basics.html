MicroPython Basics

Syntax and Structure of MicroPython:

MicroPython syntax and structure are quite similar to Python, making it easy for Python developers to transition to MicroPython. Here are some key points:

  1. Indentation: Like Python, MicroPython uses indentation to denote blocks of code. Consistent indentation is crucial for the proper functioning of the code.
  2. Comments: Comments in MicroPython are denoted by the '#' symbol, similar to Python.
  3. Case Sensitivity: MicroPython is case-sensitive. Variables and function names must be referenced with the correct capitalization.

Working with the REPL (Read-Eval-Print Loop):

The REPL (Read-Eval-Print Loop) is an interactive shell environment provided by MicroPython that allows you to execute commands and evaluate expressions directly. Here's how it works:

  1. Accessing the REPL: After connecting your MicroPython-enabled device (such as Raspberry Pi Pico) to your computer, you can access the REPL through a serial connection using terminal software like PuTTY (Windows) or screen (macOS/Linux).
  2. Interactivity: Once in the REPL, you can enter Python/MicroPython commands and see their output immediately. This makes it great for testing small snippets of code, experimenting with hardware peripherals, and debugging.
  3. Exiting the REPL: To exit the REPL, you can typically use a specific key combination (e.g., Ctrl + D) or command (e.g., `Ctrl + ]` in PuTTY) depending on the terminal software you're using.

Variables, Data Types, and Basic Operations:

MicroPython supports various data types and operations similar to Python:

  1. Variables: Variables are used to store data. They can hold different types of data, including integers, floats, strings, lists, tuples, dictionaries, etc.
  2. Data Types:
    • Integers: Whole numbers without decimal points.
    • Floats: Numbers with decimal points.
    • Strings: Sequences of characters enclosed within single or double quotes.
    • Lists: Ordered collections of items, which can be of different data types.
    • Tuples: Immutable ordered collections of items.
    • Dictionaries: Unordered collections of key-value pairs.
  3. Basic Operations:
    • Arithmetic Operations: Addition (+), Subtraction (-), Multiplication (*), Division (/), Exponentiation (**), Modulus (%), Floor Division (//).
    • Comparison Operators: Equal to (==), Not equal to (!=), Greater than (>), Less than (<), Greater than or equal to (>=), Less than or equal to (<=).
    • Logical Operators: AND (and), OR (or), NOT (not).

Here's an example of using MicroPython to perform basic operations:

# Variables and basic operations
x = 5
y = 3

# Arithmetic operations
addition_result = x + y
subtraction_result = x - y
multiplication_result = x * y
division_result = x / y
modulus_result = x % y

# Comparison operations
greater_than_result = x > y
less_than_result = x < y

# Logical operations
logical_and_result = (x > 0) and (y < 0)
logical_or_result = (x > 0) or (y < 0)

print(addition_result, subtraction_result, multiplication_result, division_result, modulus_result)
print(greater_than_result, less_than_result)
print(logical_and_result, logical_or_result)

Understanding these basics will provide you with a solid foundation for working with MicroPython and developing applications for embedded systems and microcontrollers.
