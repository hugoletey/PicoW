Programmation des GPIO

Introduction aux GPIO :

  1. Broches GPIO : Les broches GPIO sont les broches d'entrée/sortie numériques sur un microcontrôleur qui peuvent être contrôlées de manière indépendante pour lire des signaux numériques (entrée) ou produire des signaux numériques (sortie).
  2. Numérotation des Broches : Les broches GPIO sont généralement numérotées pour une référence facile. Sur le Raspberry Pi Pico, les broches GPIO sont numérotées de GP0 à GP28.

Contrôle des Sorties Numériques et Lecture des Entrées Numériques :

  1. Sorties Numériques:
    • Pour définir une broche GPIO en tant que sortie, vous utilisez une fonction telle que pin.value(1) ou pin.high() pour définir la broche à HIGH (3,3V) ou pin.value(0) ou pin.low() pour la définir à LOW (0V).
    • Exemple (MicroPython) : 
      from machine import Pin
import utime

broche_led = Pin(25, Pin.OUT)

while True:
    broche_led.toggle()
    utime.sleep(1)

      Dans cet exemple, la LED connectée à la broche GPIO 25 s'allumera et s'éteindra toutes les secondes.

  2. Entrées Numériques:
    • Pour lire l'état d'une broche GPIO configurée en tant qu'entrée, vous utilisez la méthode pin.value().
    • Exemple (MicroPython) : 
      from machine import Pin
import utime

broche_bouton = Pin(12, Pin.IN, Pin.PULL_DOWN)

while True:
    if broche_bouton.value() == 1:
        print("Bouton pressé !")
    utime.sleep(0.1)

      Dans cet exemple, le programme vérifie si le bouton connecté à la broche GPIO 12 est pressé. S'il l'est, il affiche un message.

Modulation de Largeur d'Impulsion (PWM) pour Contrôler les Dispositifs Analogiques :

  1. PWM en MicroPython:
    • MicroPython prend en charge le PWM via la classe machine.PWM.
    • Exemple (MicroPython) : 
      from machine import Pin, PWM

broche_pwm = Pin(15)
pwm = PWM(broche_pwm)

pwm.freq(1000)  # Définir la fréquence PWM à 1 kHz
pwm.duty_u16(32768)  # Définir le cycle de service à 50%

      Dans cet exemple, nous créons un objet PWM sur la broche GPIO 15, définissons la fréquence à 1 kHz et le cycle de service à 50%.

En maîtrisant la programmation des GPIO et le PWM, vous pouvez contrôler une large gamme de dispositifs numériques et analogiques avec votre microcontrôleur, vous permettant de construire divers projets et applications.
